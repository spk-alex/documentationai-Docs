# Matter 1.5 Clusters — Decimal IDs and Descriptions

This document lists Matter 1.5 clusters using the decimal cluster ID format commonly seen in backend paths and shadows.

For a Matter shadow path like:

```text
1/257/0
```

Interpret it as:

```text
endpoint_id / cluster_id_decimal / attribute_id_decimal
```

So `1/257/0` means:

```text
Endpoint 1 / Door Lock cluster / Attribute 0
```

> Note: This table is intended for backend/device-shadow documentation. Vendor-specific, test-only, or implementation-specific clusters may appear in SDK data models but should not usually be exposed as normal production device functionality.
