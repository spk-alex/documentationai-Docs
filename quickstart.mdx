## Overview

SmartphoneKey modernizes access control for multifamily properties and short-term rentals. Residents and guests unlock doors contactlessly using Apple Wallet or Google Wallet—no app required. You achieve enterprise-grade security with easy installation and minimal maintenance.

This quickstart guides you through hardware setup, account creation, user access assignment, and testing. Complete the process in under 30 minutes for your first functional door.

<Callout kind="tip">
Prepare these prerequisites:
- SmartphoneKey Bridge hardware kit (includes lock, bridge, and power adapter)
- Smartphone with Apple Wallet (iOS 14+) or Google Wallet (Android 8+)
- Stable Wi-Fi network (`2.4GHz` band recommended)
- Property manager email for account signup
</Callout>

## Hardware Installation

Install the SmartphoneKey Bridge and lock in minutes. Mount at standard deadbolt height (`{38-42 inches}` from floor).

<Steps>
  <Step title="Unbox and Mount Lock" icon="package">
    Remove the SmartphoneKey Lock from packaging. Align with your door's deadbolt hole. Secure using provided screws and template.

    ```
    Drill guide:
    - Top hole: `{1.5in}` from edge
    - Bottom hole: `{3.5in}` from edge
    ```
  </Step>

  <Step title="Connect Bridge" icon="wifi">
    Plug the Bridge into power and your `{2.4GHz}` Wi-Fi router via Ethernet (preferred) or Wi-Fi setup.

    Scan QR code on Bridge with your phone's camera to join network `SmartphoneKey-XXXX`.

    ```bash
    # Default Wi-Fi credentials (change after setup)
    SSID: SmartphoneKey-Setup
    Password: setup1234
    ```
  </Step>

  <Step title="Pair Lock to Bridge" icon="link">
    Press the Bridge sync button for `{5 seconds}` until LED flashes blue. Hold lock's reset button until it pairs (LED solid green).
  </Step>
</Steps>

## Account Setup and Onboarding

Create your dashboard account to manage properties and passes.

<Tabs>
  <Tab title="Web Dashboard" icon="globe">
    Visit `https://dashboard.smartphonekey.com` and sign up.

    ```javascript
    // Example signup payload (via browser fetch)
    const signupData = {
      email: "manager@yourproperty.com",
      propertyName: "Maple Apartments",
      bridgeSerial: "SKB-123456789"
    };
    ```

    Verify email and add your Bridge serial number from the device label.
  </Tab>

  <Tab title="Mobile Browser" icon="smartphone">
    Open `dashboard.smartphonekey.com` in mobile Safari/Chrome. Tap "Sign Up" and follow prompts.

    <Callout kind="info">
      Mobile setup auto-detects nearby Bridges via Bluetooth.
    </Callout>
  </Tab>
</Tabs>

## Assign Initial User Access

Generate and assign NFC passes to test users. Passes sync instantly to Wallets.

<CodeGroup tabs="Dashboard API,cURL">
  ```javascript
  // Dashboard API: Create pass
  const response = await fetch('https://api.smartphonekey.com/v1/passes', {
    method: 'POST',
    headers: { 'Authorization': 'Bearer YOUR_API_KEY' },
    body: JSON.stringify({
      userEmail: 'resident@tenant.com',
      propertyId: 'prop_abc123',
      doors: ['front-door-1']
    })
  });
  ```
  ```bash
  curl -X POST https://api.smartphonekey.com/v1/passes \
    -H "Authorization: Bearer YOUR_API_KEY" \
    -d '{
      "userEmail": "resident@tenant.com",
      "propertyId": "prop_abc123",
      "doors": ["front-door-1"]
    }'
  ```
</CodeGroup>

Users receive an email with Wallet add instructions. Passes auto-update on access changes.
